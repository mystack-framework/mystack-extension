# MyStack Framework Helper

**The Ultimate, Auto-Healing, Single-Class Oriented PHP Framework Extension.**

Welcome to the official VS Code extension for the **MyStack Framework**, developed by **Sakibur Rahman (@sakibweb)**. This extension is designed to supercharge your development workflow by providing intelligent autocompletion, powerful code snippets, and in-editor documentation for all MyStack core components.

## 🚀 Features

- **Massive Snippet Library:** Autocomplete for all core classes (`PHDB`, `PHRO`, `PHUI`, `PHPA`, `PHJS`, `PHCS`, `PHEM`, `PHEV`, `PHAU`, etc.).
- **Intelligent Hints:** Hover over functions to read highly professional, descriptive docblocks explaining exactly what each function does (e.g., `PHDB::select`, `PHPA::bkash()->charge()`).
- **Tailwind Utility CSS IntelliSense:** Direct support for `PHCS` utility classes. Type `class:bg-` or `class:flex` to see instant suggestions.
- **HTML5 Tags Autocomplete:** Type `html:div` or `html:button` to instantly generate perfectly formatted standard HTML5 tags.
- **UI Component Rendering:** Access 7000+ native PHP UI components simply by typing `ui:` (e.g., `ui:section:hero`).
- **Dynamic NLP JS (PHJS):** Snippets for `phjs()` and `tjs()` to seamlessly integrate human-readable Alpine.js and HTMX logic.

## 🛠️ Usage

Simply open any `.php` or `.html` file inside your MyStack project and start typing:

- Type `phdb:` or `db:` to get Database suggestions.
- Type `phro:` or `route:` for Router suggestions.
- Type `ui:` for UI components.
- Type `phpa:` to explore payment gateways.
- Type `class:` for Tailwind-like CSS utilities.
- Type `html:` for standard tags.
